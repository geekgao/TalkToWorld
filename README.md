# TalkToWorld
为中文地区渐冻症、失语者制作的自动发音键盘(Automatic pronunciation keyboard for people with ALS and aphasia in Chinese)
## 原理
- 利用系统自带的TTS朗读输入文字
- 利用Google中文输入接口获取可能的输入词
